## Contribute

``` bash
# install dependencies
yarn

# rebuild sqlite3 
yarn run rebuild # !!!!!important!!!!!

# serve with hot reload at localhost:9080
yarn run dev 

# build electron application for production
yarn run build 

# run unit tests
yarn test 

# lint all JS/Vue component files in `src/`
yarn run lint 

```
